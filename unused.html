Unused Content

Soup 0.9 has a rather decent amount of unused content in its files, ranging from unused graphics to music which never made the cut.

Graphics

Day 6, Day 7 and Exit

The game's Day Select sheet contains unused sprites for Days 6 and 7, as well as an Exit button.

Its likely that the game was supposed to have a full playable week (7 days) instead of 5, as well as an Exit button serving as an another way to leave the game from the menu aside from pressing the Esc key, though this is mere speculation as it was never confirmed.

title5.gif

A fourth variation of the game's Title Cards which never shows up ingame. It has the big green number 10 from the title overlay, though without the japanese text reading "Stereo".

tarroom4's Object Variations

The Telephone Room from Day 5 features 3 additional variations of the room's object which go unused.

All 4 variations of the object seem to rudely tell the player to stop playing the game for an unknown reason.

Its possible that the game would pick one of the 4 versions of the object randomly each time the player entered the room, however once again this was never confirmed, thus remains only a speculation.

tarroom1's Additional Overlay

The Cubic Room from Day 2 has an extra, slightly brighter version of its Overlay. Its difficult to tell what purpose it could have played if it was used.

fontchr.bmp

The FPS Counter uses this font graphic. However, it only uses the numbers, every other symbol remains unused.

Music

don01

don01.ogg is an unused track in the game, none of the rooms use it, the track wind.ogg is used twice instead: by the Poem Room and the Corrupted Drawing Room from Day 4 and Day 5 respectively.
