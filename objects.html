Objects

Each room in Soup 0.9 contains an object in the center. They are static images which always face the camera.

Objects are internally referred as "chr" (short for Character), so they can be also called Characters alongside Mini Characters, which are referred as "minichr".

Mouth Room (donroom6) and Poem Room (donroom5) have their objects set as transparent, looking as if they don't have objects in them.

Telephone Room (tarroom4) has three variants of its object which go unused.
